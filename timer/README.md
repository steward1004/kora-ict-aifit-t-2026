포모도로 타이머와 할 일 목록을 함께 보여주는 간단한 정적 웹 페이지 예제입니다.

## 구현 내용
- 포모도로 타이머 UI
  - 시작, 일시정지, 리셋 버튼 제공
  - 작업 / 짧은 휴식 / 긴 휴식 모드 전환 지원
- 할 일 목록 UI
  - 할 일 입력 및 추가
  - 완료 처리 및 삭제
  - 로컬 스토리지에 자동 저장

## 실행 방법
1. 브라우저에서 index.html 파일을 열면 바로 확인할 수 있습니다.
2. 또는 로컬 서버로 실행할 수 있습니다.

```bash
python3 -m http.server 8000
```

그다음 브라우저에서 http://localhost:8000 으로 접속합니다.

## 파일
- index.html: 메인 페이지와 스타일, 동작 로직 포함
