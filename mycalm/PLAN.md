# MyCalm MVP 설계 계획서 (PLAN.md)

이 문서는 프로젝트 폴더 내에 기록되는 설계 계획서입니다.

## 1. 구현 목표
- 시험불안 완화 및 취침 루틴을 돕는 청소년용 웹 애플리케이션 `MyCalm` MVP 구현.
- 기술 스택: Vanilla HTML, CSS, JavaScript 및 LocalStorage.

## 2. 세부 구성 및 파일 목록
- **홈 (`index.html`)**: 시간대별 인사말, 스트릭 표시, 수능 D-Day 카운터, 메뉴 이동 카드.
- **호흡 세션 (`breathing.html`)**: 4-7-8 호흡 사이클(19초) 시각화 애니메이션 및 5분 세션 타이머.
- **Sleep Stories (`stories.html`)**: 3가지 데모 수면 유도 오디오 재생 인터페이스. 끝까지 재생 완독 시에만 기록 저장.
- **오늘의 루틴 (`routine.html`)**: 투두 형태의 수면 루틴 체크리스트, 취침 시간 알림(Notification API) 설정.
- **학부모 리포트 (`parent-report.html`)**: 4자리 PIN 인증, 이용 일수, 평균 취침 시간 및 스트릭 정보 요약 보고.
- **스타일 (`css/style.css`)**: 라벤더 및 페일 블루 테마, 글래스모피즘, 가독성 텍스트 오버레이, 반응형 디자인.
- **스크립트 (`js/`)**:
  - `storage.js`: 로컬 타임존 기반 날짜 및 로컬스토리지 제어 공통 모듈.
  - `background.js`: 안개 숲 배경 이미지 2장을 60초 주기로 페이드 교차시키는 효과.
  - `breathing.js`, `stories.js`, `routine.js`, `parent-report.js`: 개별 페이지 핵심 동작 제어.
- **리소스 (`assets/`)**:
  - 배경 안개 숲 이미지 2장 (`bg/bg1.jpg`, `bg/bg2.jpg`)
  - 데모용 수면음 WAV 파일 3개 (`audio/story01.wav` 등)

## 3. 검증 항목
1. **D-Day 계산 정확성**: 현지 로컬 날짜 기준으로 수능 D-Day 정확도 검증.
2. **호흡 애니메이션 상태**: 4초 들숨(확장) -> 7초 유지(멈춤) -> 8초 날숨(축소) 루프 매칭.
3. **재생 완료 로그 저장**: 오디오 플레이어가 끝(ended)까지 진행됐을 때만 스토리지에 완독 로그 기록 여부 검증.
4. **학부모 PIN 제어**: 최초 설정 단계 동작, 잘못된 PIN에 대한 예외 처리.
5. **로컬 서버**: `python3 -m http.server 8080` 실행 후 브라우저 상 정상 구동 확인.
